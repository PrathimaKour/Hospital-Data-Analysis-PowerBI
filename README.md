Interactive Power BI dashboard analyzing hospital operations, patient trends, doctor availability, and medical expenses across departments and cities.

# Hospital Analytics Power BI Dashboard

## Overview
This project analyzes hospital operational and financial data using Power BI.

The dashboard provides insights into:
- Patient trends
- Medical expenses
- Department performance
- Doctor availability
- Monthly expense trends

## Problem Statement
Hospitals generate large amounts of operational data daily. 
This project aims to analyze healthcare performance, patient distribution, operational workload, and financial trends using Power BI.

## Key KPIs
- Total Patients
- Total Medical Expenses
- Average Stay Days
- Total Doctors
- Number of Departments

## Dashboard Features
- Interactive Visualizations
- KPI Cards
- Slicers & Filters
- Trend Analysis
- Department-wise Analysis
- City-wise Expense Analysis

## Business Questions Solved
- Which department handles the highest patient load?
- Which city records the highest medical expenses?
- How do doctors compare against patient demand?
- What are the monthly expense trends?
- Which hospitals incur the highest operational costs?

## Tools Used
- Power BI
- DAX
- Power Query
- CSV Dataset
